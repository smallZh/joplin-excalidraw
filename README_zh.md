# Joplin-excalidraw

在Joplin中使用[excalidraw](https://github.com/excalidraw/excalidraw) 来绘图。

## 使用方法?

插件安装完成后，点击“工具栏”里的 按钮，在弹框里绘制完成后，点击“render svg”按钮，并进行保存，会生成以下内容：

```
![excalidraw](excalidraw://970d7f3995b94321827cdcb8b9e16dc0)
```
